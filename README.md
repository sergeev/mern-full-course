## Install packages back-end
- npm install
- npm i express mongoose
- npm i -D nodemon concurrently
- npm i config
- npm i bcryptjs
- npm i express-validator
- npm i jsonwebtoken
## Install packages front-end [Client folder]
- npm install
- npm i materialize-css@next
- npm i react-router-dom
### build frontend
- npm run build
## Server run 
- npm install -g serve
- serve -s build

- npm run server
